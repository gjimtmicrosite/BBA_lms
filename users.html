📂 File Naming Convention

Use the exact naming format below for every submission:

[UserID]_[ProjectCode]_[DocumentType].[extension]
Examples:
bba26-ai-03_P1_BrandAnalysis.pdf
bba26-ai-07_P2_SocialMediaAudit.xlsx
bba26-ai-01_P3_StartupPitch.pdf
bba26-ai-05_P4_BusinessDashboard.pbix
❌ Do not use spaces in file names.
❌ Do not upload ZIP files unless faculty asks.
✅ Convert reports into PDF before uploading.
